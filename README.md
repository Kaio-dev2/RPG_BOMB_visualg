# RPG_BOMB_visualg

Aqui está meu primeiro projeto do começo do meu curso técnico onde utilizei a ferramenta Visualg que é uma ferramenta que foi criada para prática de lógica de programação.
Neste código me dediquei aos conceitos padrões mais simples de lógica para o funcionamento e para adicionar um capricho utilizei dígitos ascii para o visual do sistema.

Na Lógica eu utilizei: escreval , se/senao  e o temporizador para a morvimentação dos códigos do ascii.
